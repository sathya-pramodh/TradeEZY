Go to the rag/data folder and add all the required documents

Next run langchain_loader.py by setting the DATA_PATH variable to the location of the dataset.

Create a .env file with the variable name as "HUGGINGFACE_API" and give the hugging_face api key

Then run the query_data.py using by giving the query as well as the number of context windows to be generated by using the command -

for linux - 
python3 query_data.py "<query>" --k <number_of_context_windows>

for windows -
python query_data.py "<query>" --k <number_of_context_windows>
